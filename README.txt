# Alliedcore Consultancy Ltd Website

Free static website ready to upload to GitHub and deploy on Vercel.

Files:
- index.html
- style.css

How to use:
1. Create a GitHub repository named alliedcore-website.
2. Upload index.html and style.css.
3. Import the repository into Vercel.
4. Deploy for free.
5. Add alliedcore.co.uk in Vercel Project Settings > Domains.
